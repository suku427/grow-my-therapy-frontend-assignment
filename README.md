## üìñ Project Overview

This project is a submission for the **Grow My Therapy Front-End Developer Internship**. The goal was to take a generic template (*Lilac*) and transform it into a trustworthy, high-converting digital presence for a mental health professional.

### The Challenge
1.  **Clone:** Replicate the layout, grid system, and responsiveness of the original template.
2.  **Redesign:** Apply a new visual identity (colors, fonts, imagery) suitable for a therapy practice.
3.  **Adapt:** Rewrite all copy based on Dr. Reynolds' provided profile, ensuring tone accuracy.
4.  **Innovate:** Design and build a custom "Our Office" section to build client trust.

---

## üé® Design & Theme Decisions

The original *Lilac* template was stark and minimalist. For Dr. Reynolds, I shifted the aesthetic to evoke **safety, grounding, and professional warmth**.

| Component | Choice | Rationale |
| :--- | :--- | :--- |
| **Primary Color** | `Sage Green` (#4A7C59) | Represents growth, nature, and emotional balance. |
| **Secondary Color** | `Warm Beige` (#F5F0E6) | Reduces eye strain and creates a welcoming "living room" feel. |
| **Typography** | `Merriweather` (Serif) | A classic serif font that builds trust and academic authority. |
| **Imagery** | Nature & Soft Interiors | Avoided clinical stock photos; focused on calming environments. |

## üõ†Ô∏è Key Features implemented

### 1. Fully Responsive Layout
* Built with **Tailwind CSS Grid & Flexbox**.
* Ensures a pixel-perfect experience from Desktop (1440px) down to Mobile (375px).

### 2. SEO-Optimized Copywriting
* **Targeted H1s:** Headlines focus on location ("San Francisco") and specialty ("Anxiety & Trauma").
* **Profile-Based:** All text is derived strictly from Dr. Reynolds' profile, focusing on her "Compassionate, Evidence-Based" approach.

### 3. Custom Feature: "Our Office" Section
* *Requirement:* Create a new section not in the original template.
* *Implementation:* A split-layout section showcasing the physical therapy space.
* *Why:* For therapy clients, seeing the environment beforehand reduces anxiety and builds safety before the first session.
